# Regras do Clã

## Como acessar

1. Abra o menu lateral (☰)
2. Toque em **Regras**

A tela exibe as regras do clã em formato de texto formatado. No topo da página você verá a data da última atualização e
quem fez a alteração.

---

## Lendo as regras

As regras são exibidas com formatação (negrito, listas, links clicáveis). O texto é selecionável — você pode copiar
trechos se precisar.

Links dentro das regras podem abrir o navegador externo ao serem tocados.

---

## Editando as regras (Admin)

Se você tem cargo de **Admin**, um ícone de edição aparece no canto superior direito da tela.

### Como editar

1. Toque no ícone de **editar** (lápis)
2. O campo de texto abre para edição
3. Edite o conteúdo usando **Markdown** (veja dicas abaixo)
4. Toque em **Salvar**

Ao salvar:

- O conteúdo é atualizado para todos os membros imediatamente
- Uma **notificação push** é enviada para todos os usuários informando que as regras foram atualizadas
- A data, hora e nome do admin que editou são registrados automaticamente

### Dicas de Markdown

Toque no ícone de informação (ℹ️) dentro do modo de edição para ver as dicas disponíveis.

Principais formatações:

| O que escrever | O que aparece            |
|----------------|--------------------------|
| `**texto**`    | **texto** (negrito)      |
| `*texto*`      | *texto* (itálico)        |
| `# Título`     | Título grande            |
| `## Subtítulo` | Subtítulo                |
| `- item`       | • item (lista)           |
| `1. item`      | 1. item (lista numerada) |
| `[texto](url)` | link clicável            |

---

## Notificação de atualização

Sempre que as regras forem atualizadas por um Admin, **todos os membros** recebem uma notificação push:
> *"As Regras foram atualizadas"*

Ao receber essa notificação, acesse a seção Regras para ver o que mudou.
