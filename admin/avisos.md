# Avisos e Dicas

Admins podem gerenciar os avisos e dicas exibidos para todos os membros.

## Avisos

Os avisos são informações importantes para todo o clã — anúncios, mudanças de regras, comunicados.

![Lista de avisos do clã com título e data](/img/admin-avisos-lista.png)

### Criar um aviso

1. Vá em **Avisos**
2. Toque em **+**
3. Preencha:
   - **Título** — breve e descritivo
   - **Corpo** — o conteúdo completo (suporta formatação básica)
4. Toque em **Salvar**

A data de criação é registrada automaticamente.

### Editar um aviso

1. Acesse o aviso
2. Toque em **Editar**
3. Altere o conteúdo
4. Salve — a data de edição e o autor são registrados

### Deletar um aviso

1. Acesse o aviso
2. Toque em **Deletar** e confirme

---

## Dicas

As dicas são guias e tutoriais organizados por tags para fácil consulta.

![Lista de dicas com título e tags de categoria](/img/admin-dicas-lista.png)

### Criar uma dica

1. Vá em **Dicas**
2. Toque em **+**
3. Preencha:
   - **Título**
   - **Conteúdo** — suporta Markdown (negrito, listas, links)
   - **Tags** — categorias para filtro (ex: "alistamento", "farm", "pvp")
4. Salve

### Editar ou deletar

Mesmo processo dos avisos.

---

## Regras do Clã

As regras ficam em uma seção dedicada acessível por todos os membros.

Para editar:

1. Vá em **Gestão → Regras** (ou pelo menu)
2. Edite o conteúdo (Markdown suportado)
3. Salve

::: tip Markdown
Você pode usar:
- `**negrito**` para **negrito**
- `- item` para listas
- `# Título` para títulos
- `[texto](link)` para links
:::
