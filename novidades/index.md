# Novidades

A tela de Novidades é o mural de changelog do app: mostra o que mudou em cada versão do Timaceta, para você acompanhar as melhorias e novos recursos.

## Como acessar

1. Abra o menu lateral (☰)
2. Toque em **Novidades**

::: info
O item **Novidades** só aparece no menu quando existe pelo menos uma versão com novidades publicadas e ativas. É acessível a **qualquer usuário logado**, sem exigir cargo especial.
:::

---

## Marca "NOVO"

Quando existe conteúdo publicado que você ainda não visualizou, uma marquinha **"NOVO"** aparece ao lado do item de menu **Novidades**.

Assim que você abre a tela, o app marca as novidades como vistas e a marca **some** — ela volta a aparecer somente quando uma nova versão com novidades for publicada.

---

## O que você vê na tela

A tela lista as versões do app, **da mais recente para a mais antiga**, cada uma com os destaques daquela versão: o que foi adicionado, corrigido ou melhorado.

---

## Quem publica as novidades

::: warning
Hoje **não existe uma tela dentro do app** para o Admin publicar novidades diretamente. A publicação de cada versão é feita pela equipe de desenvolvimento por fora do aplicativo, como parte do processo de lançamento de cada atualização.
:::
