# Meu Perfil

O perfil é a sua identidade dentro do app Timaceta. Ele identifica quem você é para a liderança e para as funcionalidades do sistema.

## Como acessar

1. Abra o menu lateral (ícone ☰)
2. Toque em **Perfil**

![Tela de perfil com todos os campos](/img/perfil-tela.png)

---

## Campos do perfil

### Nome completo

**Obrigatório.** Digite seu nome e sobrenome. O sistema exige pelo menos **duas palavras** (nome + sobrenome).

- Correto: `Roberto Pegoraro`
- Incorreto: `Roberto` (apenas um nome)

Esse nome é usado pela liderança para identificar você no sistema.

---

### E-mail

**Somente leitura.** Exibe o e-mail com o qual você fez login. Não é possível alterar diretamente pelo perfil — este campo é controlado pelo seu provedor de autenticação (e-mail/senha, Google ou Apple).

---

### WhatsApp

Número de telefone para contato. O campo suporta números de qualquer país com o código internacional automaticamente.

**Como preencher:**
1. Selecione a bandeira do seu país (o código é inserido automaticamente)
2. Digite o número sem o código do país
3. Exemplo Brasil: selecione 🇧🇷, depois `11 99999-9999`

Esse número pode ser usado pela liderança para entrar em contato fora do app.

---

### Informações de Hardware

![Campo de hardware expandido com contador de caracteres](/img/perfil-hardware.png)

Campo expandível (clique para abrir) onde você descreve as especificações do seu computador.

**Regras:**
- Mínimo: **20 caracteres**
- Máximo: **150 caracteres**
- Um contador `X / 150` mostra quantos caracteres você digitou
- O campo aceita múltiplas linhas

**O que colocar aqui:**

Informe as especificações relevantes para o jogo, como processador, memória RAM, placa de vídeo e conexão de internet. Isso ajuda a liderança a entender suas condições técnicas de jogo.

Exemplo:
> `Intel i5-10400F, 16GB RAM, GTX 1060 6GB, 100Mbps fibra`

---

## Perfil incompleto

Se seu perfil estiver incompleto (especialmente o **Nome completo**), um aviso em vermelho aparecerá na tela. Algumas funcionalidades administrativas ficam bloqueadas enquanto o perfil estiver incompleto.

Complete o perfil assim que possível após a ativação da sua conta.

---

## Excluir minha conta

::: danger Ação irreversível
A exclusão de conta remove **permanentemente** seu usuário e todos os dados associados do sistema. Esta ação não pode ser desfeita.
:::

Para excluir sua conta:

![Ícone de lixeira vermelho no canto superior direito do perfil](/img/perfil-deletar-btn.png)

1. Na tela de Perfil, toque no **ícone de lixeira** (vermelho) no canto superior direito
2. Uma janela de confirmação perguntará: *"Confirma exclusão da conta e de todos os dados?"*
3. Confirme a exclusão

O que é removido:
- Seu usuário no sistema Timaceta
- Seu usuário na autenticação Firebase
- Os vínculos com personagens (os personagens em si permanecem cadastrados, mas desvinculados de você)

Após excluir, você é redirecionado automaticamente para a tela de login.

> Se quiser apenas **desativar temporariamente** o acesso, peça a um Admin para desativar sua conta em vez de excluí-la.

---

## Salvando as alterações

Após editar qualquer campo:

1. Toque em **Salvar**
2. Uma confirmação aparecerá quando salvo com sucesso
3. Os dados são atualizados em tempo real para o sistema
