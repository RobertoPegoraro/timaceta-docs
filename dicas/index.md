# Dicas

A tela de Dicas reúne conteúdo educativo e informativo sobre o clã e o jogo, escrito em texto formatado pelo Admin.

## Como acessar

1. Abra o menu lateral (☰)
2. Toque em **Dicas**

::: info
Qualquer usuário logado pode acessar e ler as dicas — não é preciso ter nenhum cargo especial.
:::

O conteúdo é exibido com formatação (negrito, listas, links clicáveis). Links dentro de uma dica abrem no navegador externo ao serem tocados.

---

## Criando e editando dicas (Admin)

Somente quem tem cargo de **Admin** pode criar, editar e excluir dicas. Se você é Admin, verá os botões de adicionar, editar e excluir na tela de Dicas.

### Como criar ou editar

1. Toque no botão de **adicionar** (ou no ícone de editar de uma dica existente)
2. Preencha o título e o conteúdo, usando **Markdown** para formatar o texto
3. Toque no ícone de **ajuda** (informação) para ver as dicas de formatação disponíveis
4. Toque em **Salvar**

::: warning Perfil incompleto
Não é possível salvar uma dica se o seu próprio perfil de Admin estiver incompleto. Complete pelo menos **nome** e **telefone** no seu [perfil](/perfil/) antes de tentar salvar.
:::

### Excluindo uma dica

Toque no ícone de **excluir** na dica desejada. Essa ação não pode ser desfeita.

## Dicas de Markdown

Principais formatações aceitas no conteúdo:

| O que escrever | O que aparece |
|---|---|
| `**texto**` | **texto** (negrito) |
| `*texto*` | *texto* (itálico) |
| `# Título` | Título grande |
| `## Subtítulo` | Subtítulo |
| `- item` | • item (lista) |
| `1. item` | 1. item (lista numerada) |
| `[texto](url)` | link clicável |
