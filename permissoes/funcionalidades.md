# O que cada cargo pode fazer

Consulte a [página de Cargos](/permissoes/) para a tabela completa de permissões.

Esta página descreve em detalhe as funcionalidades exclusivas de cada cargo.

## Funcionalidades exclusivas de Admin

### Gestão de Clãs

- Cadastrar novos clãs com suas credenciais do Painel
- Editar informações (nome, servidor, credenciais, nível mínimo)
- Ativar/desativar o alistamento automático por clã
- Ativar/desativar a exigência de voto no BC por clã
- Deletar clãs
- Resetar sessão de login (forçar novo login no próximo ciclo)

### Gestão de Usuários

- Ver todos os usuários cadastrados
- Ativar ou desativar acesso de um usuário
- Atribuir ou remover cargos (Recrutador, Coordenador, Gerenciador etc.)
- Ver último acesso de cada usuário

### Configurações Globais

- Ativar/desativar criação de contas públicas
- Ativar/desativar o processo de recrutamento
- Definir nível mínimo para recrutamento
- Ativar/desativar o horário de boss
- Configurar link do Instagram
- Ativar/desativar o alistamento manual

### Conteúdo

- Criar, editar e deletar avisos
- Criar, editar e deletar dicas
- Editar as regras do clã
- Gerar lista de personagens
- Gerar relatório de participações em BC

### Notificações

- Enviar notificação para todos os usuários
- Enviar para grupos específicos
- Enviar para usuários com perfil incompleto
- Enviar para quem nunca acessou o app

---

## Funcionalidades do Recrutador

- Dashboard especial de recrutamento
- Criar novo candidato
- Editar informações do candidato
- Ativar votação (notifica membros automaticamente)
- Encerrar votação
- Ver lista completa de votos e comentários
- Aprovar ou rejeitar com motivo
- Receber notificação push para novo candidato

---

## Funcionalidades do Coordenador BC

- Criar evento de Bless Castle com data/hora/descrição
- Editar evento enquanto ativo
- Encerrar evento (muda para inativo — usado como referência no alistamento)
- Ver contagem de presença (SIM/NÃO/SEM RESPOSTA)
- Ver lista completa de quem votou

---

## Funcionalidades do Coordenador PvP

- Criar evento de PvP (com Clã Primário e, opcionalmente, Clã Secundário)
- Editar e ativar/desativar eventos PvP
- Ver a tela de Confirmações (todos os votos, com filtro Sim/Não/Todos)
- Criar, editar e **excluir** qualquer grupo PvP (Normal ou Mágico) do evento
- Designar ou alterar o piloto de qualquer personagem em qualquer grupo
- Exportar a lista de grupos formatada (compartilhar via WhatsApp)

::: info Criar grupos não é exclusivo do Coordenador
Qualquer membro pode criar um grupo PvP e gerenciá-lo (adicionar/remover personagens, designar pilotos). A diferença do
Coordenador/Admin é que eles gerenciam **todos** os grupos do evento (não só os que criaram) e são os únicos que podem
**excluir** um grupo. Veja [PvP — Votação, Grupos e Pilotos](/eventos/grupos-pvp).
:::

---

## Funcionalidades do Gerenciador de Clã

- Acessar a lista atualizada de membros do clã (dados do Painel)
- Ver: posição, nick, classe, nível, data de entrada, último acesso
- Aceitar alistados manualmente (sem precisar ir ao site Painel)
- Rejeitar alistados
- Remover membros do clã

::: warning
A remoção de membros pelo app é **imediata e irreversível** no Painel. Use com cuidado.
:::
