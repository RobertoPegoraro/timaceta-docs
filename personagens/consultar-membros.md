# Consultar Membros

A funcionalidade **Consultar Membros** permite buscar personagens cadastrados pelo nome e ver quem é o responsável por eles no clã.

## Como acessar

1. Abra o menu lateral (☰)
2. Toque em **Consultar Membros**

---

## Como usar

1. Digite **parte do nick** do personagem no campo de busca — não precisa ser o nome completo nem exato
2. Pressione **Enter** (ou a tecla de busca do teclado) para disparar a pesquisa

::: info Sem ícone de lupa
Não existe um botão de lupa para iniciar a busca — ela é feita ao apertar Enter/a tecla de busca do teclado. Enquanto o campo não estiver vazio, aparece apenas um botão de **limpar** (✕) para apagar o texto digitado.
:::

A busca **não diferencia maiúsculas de minúsculas** — você pode digitar `knightrobert` e encontrar `KnightRobert`.

---

## Resultado da busca

Como a busca é por **parte do nome**, ela pode retornar **vários personagens ao mesmo tempo**, cada um exibido em um card próprio — não é um resultado único de "encontrado/não encontrado".

Cada card mostra:

- O nick do personagem
- Se o usuário vinculado está ativo no sistema
- O **telefone de contato (WhatsApp)** do dono do personagem

::: warning Contato exibe dado pessoal de outra pessoa
Cada card traz um link que abre diretamente uma conversa no WhatsApp com o dono do personagem, e o número também pode ser **copiado segurando o dedo sobre ele** (long-press). Use esse contato com bom senso — trata-se do telefone pessoal de outro membro do clã, exposto para facilitar a comunicação entre jogadores.
:::

Se nenhum personagem corresponder ao texto digitado, nenhum card é exibido.

---

## Para que serve

- Encontrar rapidamente personagens do clã pelo nome (ou parte dele)
- Descobrir e contatar por WhatsApp o dono de um personagem
- Verificar se um nick está registrado antes de convidá-lo para eventos
- Checar se um candidato já tem cadastro no sistema
