# Gerenciar Candidatos (Recrutador / Admin)

Esta seção é para **Recrutadores** e **Admins** que gerenciam o processo de recrutamento.

## Criar um novo recrutamento

1. Vá em **Recrutamento**
2. Toque no botão **+** (adicionar)
3. Preencha os dados do candidato baseado no formulário que ele preencheu
4. Toque em **Salvar**

Ao criar, uma notificação é enviada automaticamente para todos os **Recrutadores**.

## Ativar a votação

Após criar, o recrutamento fica com votação **inativa**. Para abrir para votos:

1. Acesse o recrutamento
2. Toque em **Ativar Votação**
3. Uma notificação é enviada para o grupo configurado

::: tip
Ative a votação somente quando tiver as informações completas do candidato. Uma vez ativa, os membros começarão a votar.
:::

## Encerrar a votação

Quando quiser encerrar as votações:

1. Acesse o recrutamento
2. Toque em **Encerrar Votação**
3. O recrutamento passa para status "inativo"

## Ver os votos

1. Acesse o recrutamento
2. Toque em **Ver Votos**
3. Você verá:
   - Contagem: SIM / NÃO / SEM POSIÇÃO / SEM VOTO
   - Lista completa de quem votou, quando e o quê
   - Comentários opcionais

## Tomar a decisão final

Após encerrar a votação:

1. Acesse o recrutamento
2. Toque em **Aprovar** ou **Rejeitar**
3. Se rejeitar: um campo de **motivo** é exibido (opcional mas recomendado)
4. Confirmação envia notificação com o resultado

## Editar um recrutamento

Para corrigir informações:

1. Acesse o recrutamento
2. Toque em **Editar**
3. Altere os campos necessários
4. Salve

## Deletar um recrutamento (Admin)

::: warning
Apenas Admins podem deletar. Ao deletar, todos os votos associados também são removidos permanentemente.
:::

1. Acesse o recrutamento
2. Toque em **Deletar**
3. Confirme a ação

## Campos do Formulário de Candidatura

O formulário completo preenchido pelo candidato contém:

| Campo | Tipo |
|---|---|
| Nome completo | Texto |
| E-mail e telefone | Contato |
| Idade e cidade | Dados pessoais |
| Ocupação | Dropdown |
| Nick principal | Texto |
| Nível e classe | Dados do personagem |
| É o criador do personagem? | Sim/Não + justificativa |
| Outros personagens? | Sim/Não + lista |
| Tem share/vendedor? | Sim/Não |
| Tem relíquias? | Sim/Não + lista |
| Tem amigos na tag? | Sim/Não |
| Foi indicado pela Timaceta? | Sim/Não |
| Participou de CLs? | Lista |
| Motivo de querer entrar | Texto livre |
| Participou BC/SoD? | Sim/Não |
| Disposto a usar TS3? | Sim/Não + motivo |
